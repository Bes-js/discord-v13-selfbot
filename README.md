# Kurulum & İçerik

- İlk olarak bilgisayarına [Node JS](https://nodejs.org/en/) indir.

- Daha sonra altyapıyı indirip '.env' dosyasına token=<tokeniniz> < > Koymayınız.

    - Konsola ` node . ` Yazarak Projeyi Başlatın.

	- Bu Kadar Bot Hazır
